# Steam Bulk Card Buyer

该脚本用于 Steam 徽章一键购买自动合成。

该项目使用 GPLv3 协议.

[源项目](https://github.com/panikajo/Steam-Bulk-Card-Buyer)

## 推荐使用场景

1. 喜加一挂卡获得三张卡牌，使用 x1 模式自动补全剩余卡牌，提升至等级1.
2. 然后使用 x4 mode 直升等级5.

Enjoy it!

## FAQ

- **为什么一直提示 Loading...**
- **A:** 可能为该卡牌在市场中无人出售或者网络错误。
